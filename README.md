# 🎌 Hiragana Quiz — 일본어 히라가나 퀴즈 웹앱  
### ✨ *v0.1 — 기본 뼈대 완성 버전*

일본어 히라가나 학습자를 위한 **웹 기반 퀴즈 앱**입니다.  
사용자가 원하는 **출제 행(아·카·사·타 등)** 을 자유롭게 켜고 끄며  
본인 수준에 맞추어 학습할 수 있습니다.

## 🚀 주요 기능 (Features)

- 🎯 **출제할 행 선택 가능**  
  - 아행 ~ 와행 전체 지원  
  - 원하는 조합으로 커스텀 학습  
  - "전체 선택" 버튼으로 빠른 toggle

- 🔄 **문제 방향 선택 지원**  
  - 한글 → 히라가나  
  - 히라가나 → 한글  

- 🧠 **객관식 4지선다 퀴즈**  
  - 매번 랜덤 출제  
  - 정답/오답 즉시 피드백  
  - 점수 및 정답률 자동 계산

- 📱 **반응형 UI**  
  - 모바일/태블릿/PC 화면 크기에 맞춰 자동 최적화

## 📦 설치 & 실행 방법

### ▶ 로컬에서 실행 (Windows / Mac)
1. 리포지토리 클론:
   ```
   git clone https://github.com/Graay97/Hiragana-quiz
   ```
2. 폴더에 들어가서:
   ```
   index.html
   ```
   을 더블 클릭 → 브라우저에서 바로 실행

### ▶ GitHub Pages 배포 (이미 완료됨)
현재 GitHub Pages로 자동 배포되고 있습니다.  
변경 후 푸시하면 자동 갱신됩니다.
